# @bash0816/magi-server

Magi Hub server package for Termux.

## Install

```bash
npm install -g @bash0816/magi-server
```

## Usage

```bash
magi-server start    # Start Magi Hub
magi-server stop     # Stop Magi Hub
magi-server status   # Show server status
magi-server restart  # Restart Magi Hub
```

## Requirements

- Termux (Android)
- Node.js 22+
